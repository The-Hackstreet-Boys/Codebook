## Installation

1. Install the required dependencies

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm run dev
   ```

## Building for production

1. Generate a production build

   ```sh
   npm run build
   ```

2. Start the server

   ```sh
   npm start
   ```
